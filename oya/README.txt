■ 親方ペット ドット絵の入れ方（解答後に出現）

このフォルダに 1.png 〜 16.png を置くと、解答後に「親方ペット」がドット絵で出現します。
無い番号は今まで通りSVGの親方にフォールバックします。

ファイル名＝親方番号（生成シートの並び順）:
  1.png  ガンコ親方        9.png  板金親方
  2.png  ほめ親方         10.png  パーツ親方
  3.png  説教親方         11.png  若手好き親方
  4.png  職人親方         12.png  無口親方
  5.png  電装親方         13.png  昭和親方
  6.png  エンジン親方     14.png  工具親方
  7.png  診断機親方       15.png  クール親方
  8.png  車検親方         16.png  感動親方

ゲーム内の出し分け（実装済み）:
  ・誤答時 … 症状の系統に合う「専門親方」が出て叱る
        電装系→5 / エンジン・燃料・冷却系→6 / 故障コード・センサー系→7 /
        足回り・ブレーキ系→4 / 車検・法令→8 / それ以外→3(説教)
  ・正解時 … ほめ系（2 ほめ / 11 若手好き / 16 感動）がランダムで褒める

推奨：
  ・1体ずつ「正面1ポーズ」だけを切り出す（4方向や表情集は不要）
  ・背景は透過PNG（白背景でも可）。ドット絵なので小さくてOK（拡大表示・pixelated）

クリーンな4×4グリッド（文字なし・透過・均等・1536×1024・キャラのみ）を
このフォルダに置いて「親方グリッド入れた」と言ってくれれば、
左上から右へ 1〜16 の順で16分割して上記名で保存します。
